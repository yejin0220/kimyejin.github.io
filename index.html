부동산거래 사이트

임차인과 임대인, 공인중개사 부동산 거래 주체들을 유기적으로 잇는 통합 주거 웹사이트 구축 하기 위한 웹페이지 구현하였습니다.
  • 개발환경 :
    - 운영체제 : Window OS
    - 개발도구 : Spring Framework, Visual Studio Code
    - DBMS : Oracle
    - Server : Apache Tomcat 9.0
    - Language : Java 11, JavaScript
    - 디자인툴 : Bootstrap
  • 프로젝트 시 활용해본 대표 기술:
    - Java, Spring Framework / Mybatis / JSP / JQuery / Ajax / JavaScript
  • 구현 기능 :
    - 로그인 : 일반고객 / 사업자 로그인 및 회원가입, 마이페이지, 찜 목록
    - 메인 페이지 : 숙소 검색
    - 관리자 페이지 : 공지사항 등록/조회/수정/삭제, 사업자 가입 승인, 사업자 정보 관리, 신고 리뷰 관리, 문의 채팅, 알림
    - 사업자 페이지 : 숙소 등록/수정/삭제, 예약확인, 사업자 리뷰 등록/고객 리뷰 신고
    - 상세보기 1 : 검색창, 옵션바(가격, 별점, 편의시설), API활용 숙소리스트(별점, 리뷰 수, 가격)
    - 상세보기 2 : 객실 선택, 장바구니, 결제, 지도, 리뷰 정렬
  • 담당 역할 :
    - 로그인 페이지(공지사항, 사업자 가입 승인 / 정보 관리, 신고 리뷰 관리 / 문의 내역), 예약 내역(리뷰 작성)
  • 회고 :
    - 효율적인 일정관리를 통해 빠르게 작업을 마칠 수 있어 다른 인원을 도울 수 있었고, 기술적인 부분에서 Ajax를 이용한 비동기 처리에 대해 한층 더 알 수 있었습니다.
